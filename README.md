# BETESHIG studios
Indie game studio based in Cluj-Napoca, Romania founded in 2025.

Official website for BETESHIG studios.

## Development

Rebuild `index.html` before pushing:

```bash
node build.js
```

This generates `index.html` from `template.html` and `content.json`.

## Deployment

The site is hosted on GitHub Pages, serving the static `index.html` directly from the `main` branch.
